A log server written in Go, that lets you stream out logs from the command line and access them over the web in 3 simple steps...

  • Pipe the output of your command to logout
  • Scan the QR code that shows up on your terminal
  • View live streaming logs on any device

Usage

The logout server listens on the TCP port 8888 for incoming logs. For a bare bones mechanism to stream logs, one can use netcat which comes pre-installed on most Linux/Unix systems. 

echo "Hello World!" | nc logout.my.to 8888

The piped output is streamed out to the logout server and a unique log id (along with a URL and QR code) is printed to the terminal, using which one can view the logs.
